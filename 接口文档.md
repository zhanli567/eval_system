# 接口文档

## 通用返回结构

所有接口统一返回：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| code | Integer | 业务状态码，0表示成功 |
| msg | String | 响应信息 |
| data | Object | 响应数据 |

失败示例：

```json
{
  "code": 400,
  "msg": "只有草稿版本允许修改",
  "data": null
}
```

## 评测集模块

### 分页查询评测集

`GET /api/datasets`

请求参数：

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| page | Integer | 否 | 页码，默认1 |
| size | Integer | 否 | 每页数量，默认10 |
| keyword | String | 否 | 评测集名称关键词 |

返回参数：

| data字段 | 类型 | 说明 |
| --- | --- | --- |
| records | Array\<DatasetSummary\> | 评测集列表 |
| total | Long | 总数 |
| page | Integer | 当前页 |
| size | Integer | 每页数量 |

DatasetSummary：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | String | 评测集ID |
| name | String | 名称 |
| description | String | 描述 |
| publishedVersionCount | Integer | 已发布版本数 |
| latestPublishedVersionId | String | 最新发布版本ID |
| latestItemCount | Integer | 最新发布版本数据量 |
| createdAt | String | 创建时间戳 |
| updatedAt | String | 更新时间戳 |

### 创建评测集

`POST /api/datasets`

请求参数：

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| name | String | 是 | 评测集名称 |
| description | String | 否 | 描述 |
| fields | Array\<FieldInput\> | 是 | 表结构字段 |

FieldInput：

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| id | String | 否 | 字段ID，创建时为空 |
| fieldName | String | 是 | 列名 |
| fieldType | String | 是 | 列类型：string/number/boolean |
| required | Boolean | 是 | 是否必填 |
| description | String | 否 | 字段描述 |

返回参数：`data` 为 `DatasetSummary`。

### 删除评测集

`DELETE /api/datasets/{datasetId}`

请求参数：

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| datasetId | String | 是 | 评测集ID |

返回参数：`data` 为 `null`。

返回参数：

| --- | --- | --- |

| total | Long | 总数 |
| page | Integer | 当前页 |
| size | Integer | 每页数量 |

TagSummary：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | String | 标签ID |
| tagName | String | 标签名称 |
| tagType | String | 标签类型 |
| description | String | 标签描述 |
| createdAt | String | 创建时间戳 |
| updatedAt | String | 更新时间戳 |

### 查询标签详情

`GET /api/tags/{tagId}`

请求参数：

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| tagId | String | 是 | 标签ID |

返回参数：`data` 为 `TagDetail`。

TagDetail：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | String | 标签ID |
| tagName | String | 标签名称 |
| tagType | String | 标签类型 |
| description | String | 标签描述 |
| minValue | Integer | 数字类型评分范围最小值 |
| maxValue | Integer | 数字类型评分范围最大值 |
| passThreshold | Integer | 数字类型通过阈值 |
| createdAt | String | 创建时间戳 |
| updatedAt | String | 更新时间戳 |
| options | Array\<TagOption\> | 分类或布尔值选项 |

TagOption：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | String | 标签选项ID |
| tagId | String | 标签ID |
| optionName | String | 选项名称 |
| optionGroup | String | 选项分组：pass通过，fail失败 |
| displayOrder | Integer | 展示顺序 |
| createdAt | String | 创建时间戳 |
| updatedAt | String | 更新时间戳 |

### 创建标签

`POST /api/tags`

请求参数：

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| tagName | String | 是 | 标签名称 |
| tagType | String | 是 | 标签类型：category/boolean/number/text |
| description | String | 否 | 标签描述 |
| minValue | Integer | 数字类型必填 | 数字类型评分范围最小值 |
| maxValue | Integer | 数字类型必填 | 数字类型评分范围最大值 |
| passThreshold | Integer | 数字类型必填 | 数字类型通过阈值 |
| options | Array\<TagOptionInput\> | 分类类型必填 | 分类标签选项；布尔值由系统固定生成True/False |

TagOptionInput：

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| id | String | 否 | 标签选项ID，创建时为空 |
| optionName | String | 是 | 选项名称 |
| optionGroup | String | 是 | 选项分组：pass/fail |

返回参数：`data` 为 `TagDetail`。

### 修改标签

`PUT /api/tags/{tagId}`

请求参数：

| 参数/字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| tagId | String | 是 | 标签ID |
| tagName | String | 是 | 标签名称 |
| tagType | String | 是 | 标签类型，不允许修改为其他类型 |
| description | String | 否 | 标签描述 |
| minValue | Integer | 数字类型必填 | 数字类型评分范围最小值 |
| maxValue | Integer | 数字类型必填 | 数字类型评分范围最大值 |
| passThreshold | Integer | 数字类型必填 | 数字类型通过阈值 |
| options | Array\<TagOptionInput\> | 分类类型必填 | 分类标签选项 |

返回参数：`data` 为 `TagDetail`。

| createdAt | String | 创建时间戳 |
| updatedAt | String | 更新时间戳 |

## 评估器模块

### 分页查询自定义评估器

`GET /api/evaluators`

请求参数：
| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| page | Integer | 否 | 页码，默认1 |
| size | Integer | 否 | 每页数量，默认10 |
| evaluatorType | String | 否 | 评估器类型：llm/code |
| keyword | String | 否 | 评估器名称关键词 |

返回参数：`data` 为 `PageResponse<EvaluatorSummary>`。

EvaluatorSummary：
| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | String | 自定义评估器ID |
| evaluatorName | String | 评估器名称 |
| evaluatorType | String | 评估器类型：llm/code |
| latestVersionId | String | 最新版本ID |
| latestVersionNo | Integer | 最新版本号：0草稿，>0发布版本 |
| latestVersionName | String | 最新版本名称：草稿/V1/V2 |
| description | String | 描述 |
| createdAt | String | 创建时间戳 |
| updatedAt | String | 更新时间戳 |

### 查询预置评估器分类

`GET /api/evaluators/presets/categories`

返回参数：`data` 为 `Array<PresetCategory>`。

PresetCategory：
| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | String | 分类ID |
| categoryName | String | 分类名称 |
| displayOrder | Integer | 展示顺序 |

### 分页查询预置评估器

`GET /api/evaluators/presets`

请求参数：
| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| page | Integer | 否 | 页码，默认1 |
| size | Integer | 否 | 每页数量，默认12 |
| categoryId | String | 否 | 分类ID，为空表示全部分类 |
| keyword | String | 否 | 预置评估器名称关键词 |

返回参数：`data` 为 `PageResponse<PresetEvaluatorSummary>`。

### 查询预置评估器详情

`GET /api/evaluators/presets/{presetId}`

返回参数：`data` 为 `PresetEvaluatorDetail`，包含描述、Prompt或执行函数、评估器参数、评分范围和通过阈值。

### 创建自定义评估器

`POST /api/evaluators`

请求参数：
| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| evaluatorName | String | 是 | 评估器名称 |
| evaluatorType | String | 是 | 评估器类型：llm/code |
| description | String | 否 | 描述 |
| modelId | String | LLM类型使用 | 模型ID，后续可作为模型表外键 |
| prompt | String | LLM类型必填 | Prompt内容 |
| executeCode | String | Code类型必填 | Python执行函数 |
| scoreMin | Decimal | 是 | 评分范围最小值 |
| scoreMax | Decimal | 是 | 评分范围最大值 |
| passThreshold | Decimal | 是 | 通过阈值 |
| params | Array<EvaluatorParam> | 否 | 评估器参数配置。LLM类型会以后端从Prompt中的`${param}`提取结果为准，前端可传入同名参数补充类型、必填和描述；Code类型为手动维护的入参配置 |

返回参数：`data` 为 `EvaluatorConfig`。新建评估器会生成草稿版本。

EvaluatorParam：
| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | String | 参数ID |
| paramName | String | 变量名 |
| dataType | String | 数据类型：string/number/boolean |
| defaultValue | String | 默认值 |
| required | Boolean | 是否必填 |
| description | String | 参数描述 |
| displayOrder | Integer | 展示顺序 |

### 复制自定义评估器

`POST /api/evaluators/{evaluatorId}/copy`

返回参数：`data` 为复制后新评估器的 `EvaluatorConfig`。

### 删除自定义评估器

`DELETE /api/evaluators/{evaluatorId}`

返回参数：`data` 为 `null`。

### 查询自定义评估器版本

`GET /api/evaluators/{evaluatorId}/versions`

返回参数：`data` 为 `Array<EvaluatorVersion>`。

EvaluatorVersion：
| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | String | 版本ID |
| evaluatorId | String | 自定义评估器ID |
| versionNo | Integer | 版本号：0草稿，>0发布版本 |
| versionName | String | 版本名称：草稿/V1/V2 |
| draft | Boolean | 是否草稿 |
| createdAt | String | 创建时间戳 |
| updatedAt | String | 更新时间戳 |

### 查询评估器版本详情

`GET /api/evaluators/versions/{versionId}`

返回参数：`data` 为 `EvaluatorConfig`，包含评估器基础信息、当前版本配置和评估器参数。LLM类型参数由Prompt中的`${param}`提取，用于评测任务绑定评测集列或待测智能体输出列。

### 修改草稿版本

`PUT /api/evaluators/versions/{versionId}`

请求参数同创建自定义评估器。仅草稿版本允许修改。

返回参数：`data` 为保存后的 `EvaluatorConfig`。

### 发布草稿版本

`POST /api/evaluators/{evaluatorId}/publish`

返回参数：`data` 为新发布版本的 `EvaluatorConfig`。发布版本只读，不允许修改。

## 评测任务模块

### 分页查询评测任务

`GET /api/tasks`

请求参数：

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| page | Integer | 否 | 页码，默认1 |
| size | Integer | 否 | 每页数量，默认10，最大100 |
| status | String | 否 | 评测状态：pending/running/completed/terminated/failed |
| keyword | String | 否 | 任务名称关键词 |
| sortBy | String | 否 | 排序字段：createdAt/updatedAt，默认updatedAt |
| sortOrder | String | 否 | 排序方向：asc/desc，默认desc |

返回参数：`data` 为 `PageResponse<TaskSummary>`。`TaskSummary` 包含任务基础信息、评估器维度概要和标签维度概要。

### 创建评测任务

`POST /api/tasks`

请求参数：

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| taskName | String | 是 | 任务名称 |
| description | String | 否 | 描述 |
| datasetId | String | 是 | 评测集ID |
| datasetVersionId | String | 是 | 评测集发布版本ID |
| appType | String | 是 | 应用类型：none不关联应用，agent智能体 |
| appId | String | agent时必填 | 智能体应用ID，当前预留 |
| appVersionId | String | agent时必填 | 智能体应用版本ID，当前预留 |
| appFieldMappings | Array | 否 | 智能体入参到评测集字段的映射，当前预留 |
| evaluators | Array | 是 | 评估器配置，1-5个 |
| tagIds | Array\<String\> | 是 | 标签ID，1-5个 |

评估器配置字段：

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| evaluatorSource | String | 是 | preset预置评估器，custom自定义评估器 |
| evaluatorId | String | 是 | 评估器ID |
| evaluatorVersionId | String | custom必填 | 自定义评估器版本ID |
| paramMappings | Array | 否 | 评估器参数映射 |

参数映射字段：

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| paramId | String | 否 | 评估器参数ID |
| paramName | String | 是 | 参数名 |
| sourceType | String | 是 | dataset_field评测集字段，app_output应用输出 |
| datasetFieldId | String | sourceType为dataset_field时必填 | 评测集字段ID |
| appOutputName | String | 否 | 应用输出字段名，单一输出可为空 |

返回参数：`data` 为 `TaskDetail`。

### 查询评测任务详情

`GET /api/tasks/{taskId}`

请求参数：

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| page | Integer | 否 | 数据明细页码，默认1 |
| size | Integer | 否 | 每页数量，默认10 |

返回参数：`data` 为 `TaskDetail`，包含基础信息、评测集字段、评估器/标签维度和任务数据行分页。

### 开始评测任务

`POST /api/tasks/{taskId}/start`

说明：当前版本未接入真实智能体和评估执行器，后端会生成本地模拟评估结果；人工标签仍需在标注页完成。

返回参数：`data` 为更新后的 `TaskDetail`。

### 终止评测任务

`POST /api/tasks/{taskId}/terminate`

返回参数：`data` 为更新后的 `TaskDetail`。

### 删除评测任务

`DELETE /api/tasks/{taskId}`

返回参数：`data` 为 `null`。任务采用软删除。

### 查询人工标注页

`GET /api/tasks/{taskId}/items/{taskItemId}/annotation`

返回参数：`data` 为 `AnnotationDetail`，包含当前数据行的评测集字段值、应用输出、标签配置、当前标注结果、自动评估器结果以及上一条/下一条任务数据行ID。

### 保存人工标注

`PUT /api/tasks/{taskId}/items/{taskItemId}/annotation`

请求参数：

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| tags | Array | 是 | 标注结果列表 |

标注结果字段：

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| taskTagId | String | 是 | 任务标签ID |
| valueText | String | text标签必填 | 文本标注值 |
| valueNumber | Decimal | number标签必填 | 数字标注值 |
| tagOptionId | String | category/boolean标签必填 | 选中的标签选项ID |

返回参数：`data` 为保存后的 `AnnotationDetail`。后端会根据标签定义计算 `passResult`。
