# 接口文档

## 通用返回结构

所有接口统一返回：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| code | Integer | 业务状态码，0表示成功 |
| msg | String | 响应信息 |
| data | Object | 响应数据 |

失败示例：

```json
{
  "code": 400,
  "msg": "只有草稿版本允许修改",
  "data": null
}
```

## 评测集模块

### 分页查询评测集

`GET /api/datasets`

请求参数：

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| page | Integer | 否 | 页码，默认1 |
| size | Integer | 否 | 每页数量，默认10 |
| keyword | String | 否 | 评测集名称关键词 |

返回参数：

| data字段 | 类型 | 说明 |
| --- | --- | --- |
| records | Array\<DatasetSummary\> | 评测集列表 |
| total | Long | 总数 |
| page | Integer | 当前页 |
| size | Integer | 每页数量 |

DatasetSummary：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | String | 评测集ID |
| name | String | 名称 |
| description | String | 描述 |
| publishedVersionCount | Integer | 已发布版本数 |
| latestPublishedVersionId | String | 最新发布版本ID |
| latestItemCount | Integer | 最新发布版本数据量 |
| createdAt | String | 创建时间戳 |
| updatedAt | String | 更新时间戳 |

### 创建评测集

`POST /api/datasets`

请求参数：

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| name | String | 是 | 评测集名称 |
| description | String | 否 | 描述 |
| fields | Array\<FieldInput\> | 是 | 表结构字段 |

FieldInput：

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| id | String | 否 | 字段ID，创建时为空 |
| fieldName | String | 是 | 列名 |
| fieldType | String | 是 | 列类型：string/number/boolean |
| required | Boolean | 是 | 是否必填 |
| description | String | 否 | 字段描述 |

返回参数：`data` 为 `DatasetSummary`。

### 删除评测集

`DELETE /api/datasets/{datasetId}`

请求参数：

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| datasetId | String | 是 | 评测集ID |

返回参数：`data` 为 `null`。

## 标签管理模块

### 分页查询标签

`GET /api/tags`

请求参数：

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| page | Integer | 否 | 页码，默认1 |
| size | Integer | 否 | 每页数量，默认10，最大100 |
| tagType | String | 否 | 标签类型：category分类，boolean布尔值，number数字，text文本 |
| keyword | String | 否 | 标签名称关键词 |

返回参数：

| data字段 | 类型 | 说明 |
| --- | --- | --- |
| records | Array\<TagSummary\> | 标签列表 |
| total | Long | 总数 |
| page | Integer | 当前页 |
| size | Integer | 每页数量 |

TagSummary：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | String | 标签ID |
| tagName | String | 标签名称 |
| tagType | String | 标签类型 |
| description | String | 标签描述 |
| createdAt | String | 创建时间戳 |
| updatedAt | String | 更新时间戳 |

### 查询标签详情

`GET /api/tags/{tagId}`

请求参数：

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| tagId | String | 是 | 标签ID |

返回参数：`data` 为 `TagDetail`。

TagDetail：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | String | 标签ID |
| tagName | String | 标签名称 |
| tagType | String | 标签类型 |
| description | String | 标签描述 |
| minValue | Integer | 数字类型评分范围最小值 |
| maxValue | Integer | 数字类型评分范围最大值 |
| passThreshold | Integer | 数字类型通过阈值 |
| createdAt | String | 创建时间戳 |
| updatedAt | String | 更新时间戳 |
| options | Array\<TagOption\> | 分类或布尔值选项 |

TagOption：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | String | 标签选项ID |
| tagId | String | 标签ID |
| optionName | String | 选项名称 |
| optionGroup | String | 选项分组：pass通过，fail失败 |
| displayOrder | Integer | 展示顺序 |
| createdAt | String | 创建时间戳 |
| updatedAt | String | 更新时间戳 |

### 创建标签

`POST /api/tags`

请求参数：

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| tagName | String | 是 | 标签名称 |
| tagType | String | 是 | 标签类型：category/boolean/number/text |
| description | String | 否 | 标签描述 |
| minValue | Integer | 数字类型必填 | 数字类型评分范围最小值 |
| maxValue | Integer | 数字类型必填 | 数字类型评分范围最大值 |
| passThreshold | Integer | 数字类型必填 | 数字类型通过阈值 |
| options | Array\<TagOptionInput\> | 分类类型必填 | 分类标签选项；布尔值由系统固定生成True/False |

TagOptionInput：

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| id | String | 否 | 标签选项ID，创建时为空 |
| optionName | String | 是 | 选项名称 |
| optionGroup | String | 是 | 选项分组：pass/fail |

返回参数：`data` 为 `TagDetail`。

### 修改标签

`PUT /api/tags/{tagId}`

请求参数：

| 参数/字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| tagId | String | 是 | 标签ID |
| tagName | String | 是 | 标签名称 |
| tagType | String | 是 | 标签类型，不允许修改为其他类型 |
| description | String | 否 | 标签描述 |
| minValue | Integer | 数字类型必填 | 数字类型评分范围最小值 |
| maxValue | Integer | 数字类型必填 | 数字类型评分范围最大值 |
| passThreshold | Integer | 数字类型必填 | 数字类型通过阈值 |
| options | Array\<TagOptionInput\> | 分类类型必填 | 分类标签选项 |

返回参数：`data` 为 `TagDetail`。

### 查询评测集版本

`GET /api/datasets/{datasetId}/versions`

请求参数：

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| datasetId | String | 是 | 评测集ID |

返回参数：`data` 为 `Array<DatasetVersion>`。

DatasetVersion：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | String | 版本ID |
| datasetId | String | 评测集ID |
| versionNo | Integer | 版本号，0为草稿 |
| versionName | String | 版本名，草稿/V1/V2 |
| itemCount | Integer | 数据行数 |
| draft | Boolean | 是否草稿 |
| createdAt | String | 创建时间戳 |
| updatedAt | String | 更新时间戳 |

### 发布草稿版本

`POST /api/datasets/{datasetId}/publish`

请求参数：

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| datasetId | String | 是 | 评测集ID |

返回参数：`data` 为新发布的 `DatasetVersion`。

### 用发布版本覆盖草稿

`POST /api/datasets/{datasetId}/versions/{versionId}/cover-draft`

请求参数：

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| datasetId | String | 是 | 评测集ID |
| versionId | String | 是 | 来源发布版本ID |

返回参数：`data` 为覆盖后的草稿 `DatasetVersion`。

### 删除发布版本

`DELETE /api/datasets/versions/{versionId}`

请求参数：

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| versionId | String | 是 | 版本ID |

返回参数：`data` 为 `null`。草稿版本不允许删除。

### 查询版本详情和数据

`GET /api/datasets/versions/{versionId}`

请求参数：

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| versionId | String | 是 | 版本ID |
| page | Integer | 否 | 页码，默认1 |
| size | Integer | 否 | 每页数量，默认10 |
| fieldId | String | 否 | 搜索字段ID |
| keyword | String | 否 | 搜索关键词 |

返回参数：

| data字段 | 类型 | 说明 |
| --- | --- | --- |
| version | DatasetVersion | 版本信息 |
| fields | Array\<DatasetField\> | 表头字段 |
| rows | PageResponse\<DatasetRow\> | 数据行分页 |

DatasetField：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | String | 字段ID |
| versionId | String | 版本ID |
| fieldName | String | 列名 |
| fieldType | String | 列类型 |
| required | Boolean | 是否必填 |
| description | String | 描述 |
| displayOrder | Integer | 展示顺序 |

DatasetRow：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | String | 数据行ID |
| rowNo | Integer | 行号 |
| values | Object | 单元格值，key为字段ID，value为单元格内容 |
| createdAt | String | 创建时间戳 |
| updatedAt | String | 更新时间戳 |

### 保存草稿表头

`PUT /api/datasets/versions/{versionId}/fields`

请求参数：

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| versionId | String | 是 | 草稿版本ID |
| body | Array\<FieldInput\> | 是 | 新表头字段列表 |

返回参数：`data` 为保存后的 `Array<DatasetField>`。

`POST /api/datasets/versions/{versionId}/items`

请求参数：

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| versionId | String | 是 | 草稿版本ID |
| values | Object | 是 | 单元格值，key为字段ID |

返回参数：`data` 为 `DatasetRow`。

### 批量新增数据

`POST /api/datasets/versions/{versionId}/items/batch`

请求参数：

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| versionId | String | 是 | 草稿版本ID |
| rows | Array\<Object\> | 是 | 多行数据，key为字段ID |

返回参数：`data` 为 `Array<DatasetRow>`。

### Excel导入数据

`POST /api/datasets/versions/{versionId}/items/import`

请求参数：

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| versionId | String | 是 | 草稿版本ID |
| file | MultipartFile | 是 | xlsx或xls文件，第一行表头必须和当前表头完全一致 |

返回参数：

| data字段 | 类型 | 说明 |
| --- | --- | --- |
| importedCount | Integer | 成功导入行数 |

### Excel全量覆盖数据

`POST /api/datasets/versions/{versionId}/items/import-cover`

请求参数：

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| versionId | String | 是 | 草稿版本ID |
| file | MultipartFile | 是 | xlsx或xls文件，第一行表头必须和当前表头完全一致，导入前会清空草稿原有数据 |

返回参数：

| data字段 | 类型 | 说明 |
| --- | --- | --- |
| importedCount | Integer | 覆盖导入行数 |
