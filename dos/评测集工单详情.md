# 评测集管理模块

## 需求背景

建设评测系统中的评测集管理能力，用于维护评测样本的数据结构、样本数据和版本。发布后的评测集版本可被评测任务引用，保证任务执行使用稳定、可追踪的数据快照。

## 需求范围

包含评测集列表、创建、删除、版本管理、字段管理、数据行管理、Excel 追加导入、Excel 全量覆盖导入、草稿发布、发布版本覆盖草稿。

## 业务规则

1. 评测集创建
   - 评测集名称必填，最长 50 个字符。
   - 描述非必填，最长 200 个字符。
   - 创建时必须维护表结构字段。
   - 创建成功后自动生成草稿版本，草稿版本 `versionNo = 0`。

2. 评测集列表
   - 支持分页查询。
   - 支持按评测集名称关键字搜索。
   - 展示名称、描述、已发布版本数、最新发布版本数据量、创建时间、更新时间。
   - 删除评测集为软删除，删除后列表不再展示。

3. 版本管理
   - 每个评测集有一个草稿版本。
   - 草稿版本可编辑字段和数据。
   - 发布版本只读，不允许编辑字段和数据。
   - 发布草稿时复制当前草稿字段和数据，生成新的发布版本，版本号按 V1、V2 递增。
   - 草稿版本不允许删除。
   - 发布版本允许删除。
   - 支持用发布版本全量覆盖当前草稿。

4. 字段管理
   - 字段名必填。
   - 同一版本内字段名不可重复。
   - 字段类型支持 `string`、`number`、`boolean`。
   - 字段支持必填标记、描述、展示顺序。
   - 新增字段时，已有数据行对应新增空单元格。
   - 删除字段时，同步删除该字段下的单元格数据。
   - 当前必填标记用于页面展示和 Excel 必填列校验，不对单条数据行空值做强校验。

5. 数据行管理
   - 仅草稿版本支持新增、编辑、删除数据行。
   - 数据行按 `rowNo` 升序展示。
   - 新增数据行时按当前最大行号递增。
   - 删除数据行后更新版本数据量。
   - 支持按指定字段和关键字筛选数据行。

6. Excel 导入
   - 仅支持 `.xlsx` 和 `.xls`。
   - 读取第一个 Sheet。
   - 第一行为表头。
   - Excel 表头按字段名匹配评测集字段。
   - 必填字段必须出现在 Excel 表头中。
   - 未匹配字段的 Excel 列忽略。
   - 空行跳过。
   - 同一次 Excel 导入中，完全重复的数据行去重。
   - 追加导入保留原草稿数据。
   - 全量覆盖导入会先清空草稿数据，再写入 Excel 数据。

## API 规则

后端真实接口路径不带 `/api`；前端开发环境的 `/api` 是 Vite 代理前缀。

| 方法 | 路径 | 说明 |
|---|---|---|
| GET | `/datasets` | 分页查询评测集 |
| POST | `/datasets` | 创建评测集 |
| DELETE | `/datasets/{datasetId}` | 删除评测集 |
| GET | `/datasets/{datasetId}/versions` | 查询版本列表 |
| POST | `/datasets/{datasetId}/publish` | 发布草稿 |
| POST | `/datasets/{datasetId}/versions/{versionId}/cover-draft` | 发布版本覆盖草稿 |
| DELETE | `/datasets/versions/{versionId}` | 删除发布版本 |
| GET | `/datasets/versions/{versionId}` | 查询版本详情 |
| PUT | `/datasets/versions/{versionId}/fields` | 替换草稿字段 |
| POST | `/datasets/versions/{versionId}/items` | 新增数据行 |
| POST | `/datasets/versions/{versionId}/items/batch` | 批量新增数据行 |
| POST | `/datasets/versions/{versionId}/items/import` | Excel 追加导入 |
| POST | `/datasets/versions/{versionId}/items/import-cover` | Excel 全量覆盖导入 |
| PUT | `/datasets/versions/{versionId}/items/{itemId}` | 编辑数据行 |
| DELETE | `/datasets/versions/{versionId}/items/{itemId}` | 删除数据行 |

## 原型修改

需要覆盖评测集列表页、创建评测集弹窗、评测集详情页、字段编辑弹窗、数据新增/编辑弹窗、Excel 导入入口、版本列表与发布/覆盖/删除操作入口。截图由产品侧补充。

## 验收标准

1. 可以创建评测集，创建后自动生成草稿版本。
2. 列表支持分页、搜索、刷新、进入详情、删除。
3. 草稿版本可以编辑字段、调整顺序、新增/编辑/删除数据行。
4. 发布版本只读，不展示编辑字段和数据的操作。
5. Excel 追加导入、全量覆盖导入按业务规则生效。
6. 发布草稿后生成新的发布版本，版本数据与发布时草稿一致。
7. 删除发布版本后版本列表不再展示该版本，草稿不可删除。
8. 使用发布版本覆盖草稿后，草稿字段和数据与所选发布版本一致。
9. 所有接口统一返回 `ApiResponse`，业务参数错误返回 HTTP 400。
